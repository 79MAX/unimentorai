# Politique de sécurité – UniMentorAI

## 🔒 Principes
- Toutes les données sensibles sont stockées de façon sécurisée (`flutter_secure_storage`).
- Les accès Firestore sont conditionnés par rôle et audités.
- Les secrets/API sont dans `.env` (jamais versionné).
- Les dépendances sont maintenues à jour et auditées.

## 🚨 Signalement de vulnérabilité
Si vous découvrez une faille, merci de la signaler immédiatement à l’équipe via un canal privé (email, issue GitHub en mode privé).

## 🔄 Gestion des incidents
- Correction prioritaire des failles critiques.
- Communication transparente avec les utilisateurs si besoin.

Merci de contribuer à la sécurité de UniMentorAI !
