# 1. Introduction

Bienvenue sur **UniMentorAI**.

La présente Politique de Confidentialité explique comment UniMentorAI ("UniMentorAI", "nous", "notre" ou "nos") collecte, utilise, stocke, traite et protège vos données personnelles lorsque vous utilisez nos sites web, applications mobiles, applications web, API et services associés (collectivement les « Services »).

Nous nous engageons à protéger votre vie privée et à assurer la transparence, la sécurité et la conformité avec les lois internationales de protection des données.

En utilisant nos Services, vous acceptez les pratiques décrites dans cette Politique de Confidentialité.

---

# 2. Données que nous collectons

Nous collectons différents types de données afin de fournir et améliorer nos Services.

## 2.1 Données fournies directement par l’utilisateur

Cela peut inclure :

* Nom et prénom
* Adresse email
* Mot de passe (chiffré)
* Informations de profil
* Préférences linguistiques
* Inscriptions aux cours
* Progression d’apprentissage
* Informations de paiement (traitées de manière sécurisée par des prestataires tiers)
* Messages envoyés au support

---

## 2.2 Données collectées automatiquement

Lorsque vous utilisez UniMentorAI, nous pouvons collecter :

* Informations sur l’appareil
* Adresse IP
* Type de navigateur
* Système d’exploitation
* Données d’utilisation
* Pages consultées
* Durée des sessions
* Comportement de navigation
* Données de performance
* Rapports de crash

---

## 2.3 Cookies et technologies similaires

Nous utilisons des cookies, SDK, pixels et technologies similaires pour :

* authentifier les utilisateurs ;
* mémoriser les préférences ;
* améliorer les performances de la plateforme ;
* personnaliser l’expérience d’apprentissage ;
* analyser l’utilisation ;
* renforcer la sécurité.

Pour plus d’informations, consultez notre Politique de Cookies.

---

## 2.4 Données liées à l’intelligence artificielle

UniMentorAI utilise des systèmes d’intelligence artificielle pour personnaliser l’apprentissage.

Nous pouvons traiter :

* interactions avec le tutor IA ;
* progression d’apprentissage ;
* recommandations personnalisées ;
* feedback utilisateur ;
* parcours adaptatif ;
* niveau de compétence.

Ces données servent uniquement à améliorer votre expérience éducative.

# 3. Utilisation des données

Nous utilisons vos données pour :

* fournir et maintenir la plateforme ;
* personnaliser l’expérience utilisateur ;
* générer les réponses de l’IA ;
* améliorer les recommandations de cours ;
* traiter les paiements et abonnements ;
* assurer la sécurité de la plateforme ;
* détecter les fraudes ;
* fournir un support client ;
* respecter nos obligations légales ;
* améliorer les performances du système.

---

# 4. Base légale du traitement (RGPD)

Si vous êtes situé dans l’Espace Économique Européen (EEE), nous traitons vos données sur les bases suivantes :

* votre consentement ;
* l’exécution d’un contrat ;
* le respect d’obligations légales ;
* nos intérêts légitimes (sécurité, amélioration du service, performance).

---

# 5. Partage des données

Nous ne vendons jamais vos données personnelles.

Nous pouvons partager vos données uniquement avec des prestataires tiers de confiance, notamment :

* fournisseurs de paiement (Stripe, PayPal, Kkiapay, etc.) ;
* services cloud (Firebase, AWS, etc.) ;
* outils d’analyse ;
* services d’authentification ;
* outils de support client ;
* systèmes de prévention de fraude.

Tous ces partenaires sont contractuellement tenus de protéger vos données.

---

# 6. Conservation des données

Nous conservons vos données uniquement pendant la durée nécessaire :

* à la fourniture des Services ;
* au respect des obligations légales ;
* à la résolution des litiges ;
* à l’application de nos accords.

La durée de conservation dépend du type de données et des exigences légales applicables.

---

# 7. Sécurité des données

Nous appliquons des mesures de sécurité avancées, notamment :

* chiffrement des données (en transit et au repos) ;
* contrôle d’accès strict ;
* authentification sécurisée ;
* surveillance des systèmes ;
* protection contre la fraude ;
* sécurisation des API.

Cependant, aucun système n’est totalement sécurisé.

---

# 8. Transferts internationaux

En tant que plateforme mondiale, vos données peuvent être traitées dans plusieurs pays.

Nous assurons leur protection via :

* Clauses Contractuelles Types (SCC) ;
* chiffrement ;
* minimisation des données ;
* infrastructures sécurisées ;
* conformité réglementaire.

Lorsque la loi l’exige, le consentement d’un parent ou tuteur est requis pour les utilisateurs mineurs.
