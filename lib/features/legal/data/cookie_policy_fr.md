# 2. Qu’est-ce qu’un cookie ?

Les cookies sont de petits fichiers texte stockés sur votre appareil (ordinateur, tablette ou mobile) lorsque vous visitez un site ou utilisez une application.

Ils permettent de :

* reconnaître votre appareil ;
* mémoriser vos préférences ;
* améliorer la sécurité ;
* personnaliser votre expérience d’apprentissage ;
* analyser les performances de la plateforme ;
* activer certaines fonctionnalités.

Les cookies ne contiennent généralement pas d’informations permettant de vous identifier directement, mais peuvent être liés à des données associées à votre compte.

Les technologies similaires incluent notamment :

* stockage local (Local Storage) ;
* stockage de session (Session Storage) ;
* pixels de suivi ;
* balises web (web beacons) ;
* identifiants SDK ;
* technologies équivalentes.

# 5. Types de cookies utilisés

## 5.1 Cookies strictement nécessaires

Ces cookies sont indispensables au fonctionnement de la plateforme.

Sans eux, certains services ne peuvent pas fonctionner correctement.

Exemples :

* authentification des utilisateurs ;
* connexion sécurisée ;
* gestion des sessions ;
* protection contre la fraude ;
* sécurité des comptes ;
* validation des paiements ;
* équilibrage de charge ;
* protection des API.

Ces cookies ne peuvent pas être désactivés.

---

## 5.2 Cookies fonctionnels

Ces cookies améliorent votre expérience utilisateur.

Ils permettent de mémoriser :

* la langue choisie ;
* le thème (clair/sombre) ;
* les paramètres d’accessibilité ;
* les préférences de notifications ;
* la personnalisation du tableau de bord ;
* le fuseau horaire ;
* les préférences d’apprentissage.

---

## 5.3 Cookies analytiques et de performance

Ces cookies nous permettent de comprendre comment la plateforme est utilisée.

Données collectées :

* pages consultées ;
* durée des sessions ;
* parcours utilisateur ;
* type d’appareil ;
* navigateur utilisé ;
* rapports de bugs ;
* statistiques d’apprentissage ;
* performances de la plateforme.

Ces données sont, lorsque possible, anonymisées ou agrégées.

UniMentorAI utilise l’intelligence artificielle pour personnaliser l’expérience d’apprentissage.

Ces cookies peuvent stocker :

* progression d’apprentissage ;
* cours complétés ;
* niveau de compétence ;
* recommandations personnalisées ;
* interactions avec l’IA ;
* parcours d’apprentissage adaptatif ;
* contenus récemment consultés ;

Ils permettent d’améliorer :

* la précision du tutorat IA ;

Ces cookies ne prennent pas de décisions juridiques ou automatiques à impact légal sur les utilisateurs.

Ces cookies renforcent la sécurité de la plateforme.
