# 1. CONDITIONS GÉNÉRALES D’UTILISATION (CGU)

## 1.1 Objet

Les présentes Conditions Générales d’Utilisation régissent l’accès et l’utilisation de la plateforme UniMentorAI, plateforme internationale de formation, mentorat, intelligence artificielle éducative et certification numérique.

## 1.2 Acceptation

L’utilisation de la plateforme implique l’acceptation intégrale des présentes CGU.

## 1.3 Accès à la plateforme

UniMentorAI peut suspendre, limiter ou interrompre temporairement l’accès à tout ou partie des services pour maintenance, sécurité ou évolution technique.

## 1.4 Comptes utilisateurs

L’utilisateur s’engage à :

* fournir des informations exactes ;
* sécuriser ses identifiants ;
* ne pas partager son compte ;
* respecter les lois applicables.

## 1.5 Interdictions

Sont strictement interdits :

* la fraude ;
* l’usurpation d’identité ;
* le piratage ;
* l’utilisation abusive de l’IA ;
* la diffusion de contenus haineux, violents ou illégaux ;
* la revente non autorisée des contenus.

## 1.6 Suspension de compte

UniMentorAI peut suspendre immédiatement un compte en cas de :

* fraude ;
* activité suspecte ;
* violation des CGU ;
* menace sécuritaire.

## 1.7 Disponibilité

UniMentorAI ne garantit pas une disponibilité ininterrompue des services.

## 1.8 Droit applicable

Le droit applicable sera défini selon la juridiction choisie par la société mère.

UniMentorAI peut collecter :

* nom ;
* email ;
* téléphone ;
* données pédagogiques ;
* données de paiement ;

Les données sont utilisées pour :

* fournir les services ;
* sécuriser la plateforme ;

* prévenir la fraude.

Les données sont conservées uniquement pour la durée nécessaire.

## 2.4 Sécurité

UniMentorAI met en œuvre :

* chiffrement ;
* journalisation ;
* contrôle d’accès ;
* surveillance anti-intrusion.

## 2.5 Droits des utilisateurs

Les utilisateurs peuvent demander :

* accès ;
* rectification ;
* suppression ;
* export des données.

# 4. POLITIQUE DE REMBOURSEMENT

## 4.1 Conditions

Les remboursements peuvent être accordés uniquement si :

* le service est inaccessible ;
* le paiement est dupliqué ;
* une erreur technique empêche l’accès.

## 4.2 Exclusions

Aucun remboursement ne sera accordé pour :

* progression déjà consommée ;
* certificat déjà délivré ;
* violation des CGU.
