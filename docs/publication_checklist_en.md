# Publication & Tender Checklist – UniMentorAI

## 1. Technical Prerequisites
- [ ] Production deployment validated (Flutter, Cloud Functions, Firestore, Storage)
- [ ] Stripe/Firebase Billing payment operational
- [ ] Admin panel accessible and secure
- [ ] Automated tests passed (unit, widget, integration, accessibility)

## 2. Compliance & GDPR
- [ ] Global audit report generated (`audit_report.md`)
- [ ] Consent and GDPR export log up to date
- [ ] Signed CGU/GDPR accessible
- [ ] GDPR export/deletion process operational

## 3. Assets & Dossiers to Provide
- [ ] Exported pricing grid (PDF)
- [ ] Sample PDF invoices
- [ ] Commercial presentation (PDF/PowerPoint)
- [ ] Technical documentation (EN/FR)
- [ ] Team training guide
- [ ] CSV/JSON export of subscriptions/invoices (if requested)

## 4. Tender Response Process
- [ ] Generate a custom quote with the simulator
- [ ] Prepare a complete dossier (see above)
- [ ] Check GDPR compliance and access security
- [ ] Review and validate all documents
- [ ] Obtain final validation from management/project team

## 5. Contacts & Support
- [ ] Technical support: support@unimentor.ai
- [ ] Sales contact: commercial@unimentor.ai
- [ ] Online documentation: https://unimentor.ai/docs
